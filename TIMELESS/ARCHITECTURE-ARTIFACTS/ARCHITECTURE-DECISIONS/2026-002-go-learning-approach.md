# ADR-2026-002: Подход к изучению Go

## Context
Нужно эффективно изучить основы Go для понимания разработки.

## Decision
Использовать комбинацию:

1. **Книга Довована**
2. **Tour of Go** (официальный туториал)
3. **Практические задачи** 
4. **Проект CLI утилиты** как финальная цель

## Rationale
- Tour of Go даёт официальную, актуальную информацию
- Задачи закрепляют синтаксис
- Проект даёт опыт полного цикла
