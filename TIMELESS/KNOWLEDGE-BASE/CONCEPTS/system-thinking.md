# Systems Thinking (Системное мышление)

## Definition
Методология понимания систем через их элементы, взаимосвязи и поведение.

## Key Principles
1. **Целое больше суммы частей** — система ≠ просто элементы
2. **Обратные связи** определяют поведение системы
3. **Запаздывания** создают колебания и нестабильность

## Applications in Software
1. **Архитектура:** Микросервисы как система взаимодействующих компонентов
2. **Мониторинг:** Метрики как обратные связи для системы
3. **Рефакторинг:** Понимание, как изменения в одной части влияют на всю систему
